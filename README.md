# Terraform AWS EC2 Nginx
Terraform AWS EC2 Nginx example. Remember to `destroy` it when done.

***WARNING!!! This might cause AWS charges.***

## Verify Changes

```bash
% cd terraform
% terraform init
% terraform validate
% terraform plan
```

## Apply Changes

```bash
% terraform apply
```

## Pre-Commits

Run manually:

```bash
% pre-commit run -a
```
